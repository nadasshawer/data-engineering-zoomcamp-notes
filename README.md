# data-engineering-zoomcamp-notes
This repository is a dedicated space for my technical notes and documentation as I progress through the DataTalksClub Data Engineering Zoomcamp.

#### 🎯 The Goal
The intention behind this repo is to document my learning journey in public, focusing on:
- Infrastructure as Code (IaC)
- Containerization & Orchestration
- Scalable Data Pipelines

#### 📂 What's Inside
I am sharing my notes directly from my Obsidian vault, including:
- Conceptual Breakdowns: Like the difference between a Docker Image (the snapshot) and a Container (the isolated instance)
- Technical Logic: Deep dives into Statelessness and why containers don't preserve changes when killed
- Syntax & Commands: Quick-reference guides for Bind Mounts (-v) and managing container lifecycles
- The focus is on Reproducibility, Isolation, and Portability—building data tools that work everywhere, every time
